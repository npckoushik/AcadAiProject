How to Play

Take turns placing X and O on the board. The first player to get three in a row wins!

Toggle the algorithm visualization to see how the minimax algorithm evaluates each possible move.

About the Algorithm

The minimax algorithm calculates the best possible move by simulating all possible game states:

  • Positive scores favor O
  • Negative scores favor X
  • Zero indicates a draw

The implementation includes alpha-beta pruning for efficiency and depth consideration to prefer quicker wins.

Code Documentation

Game Logic

The game uses a 3x3 grid represented as an array of 9 elements. Key functions:

  • calculateWinner(board): Checks if there's a winner
  • handleSquareClick(index): Handles player moves
  • resetGame(): Resets the game state
Minimax Algorithm

The minimax algorithm is a decision-making algorithm used for finding the optimal move:

  • minimax(board, depth, isMaximizing, alpha, beta): Recursive function that evaluates positions
  • runMinimax(board, isMaximizing): Runs minimax for each possible move and collects data for visualization
